ABOUT DELETE USER

A USER CAN BE DELETED WHEN:

  • The User can't be deleted if they have a log in System Log
    • If the User has started a session, it can't be deleted because it has an log associated to System Log
    • The Default User can't be removed
    • The User can't be deleted if it has a Division associated with it

IF THE USER IS NOT PRESENT IN ANY OF THE ABOVE, THIS USER CAN BE DELETED
